# ChainProof — Grower Evidence Inbox for Cooperatives and Exporters

Workflow-first DWV POC for DOG-118.

## What this prototype proves
- operator can capture grower, crop, location, and symptom context
- system can return a likely diagnosis with confidence and next actions
- result can be turned into a compliance-ready evidence record

## Current behavior
- mocked diagnosis logic
- local state only
- no backend required for the happy path
